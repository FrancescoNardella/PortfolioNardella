About Project

Project lasted 3 days, where we worked in a group. The goal was to create an e-commerce site (with home and cart). We used all our knowledge of html, css and JavaScipt (We learned the basics of JavaScript, with this project). Our idea was to create an e-commerce on the sale of musical instruments. We used Atom as editor, because it is very versatile. I took care of the html and Javascript part, learning how to use them without using frameworks, such as Bootstrap or Materialize.
